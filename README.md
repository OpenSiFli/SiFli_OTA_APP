This demo demonstrates SiFli Solution OTA and SiFli SDK OTA
## Features
### 1. Push GUIBuilder App Resource
  Home Page
### 2. SiFli Solution 2.0 OTA 
   Home -> 设置(Right Top)->开发者模式->OTA V3
### 3. SiFli-SDK OTA
   Home -> 设置(Right Top)->开发者模式->SiFli-SDK OTA
